# Home Automation using Arduino and HC-05
## Controls:
* U: Roll up the curtain
* D: Roll down the curtain
* 1: Turn on the lamp
* 2: Turn off the lamp
* R: Receive room temperature and humidity
* T: Test whether the commands are being received or not
